# z_blogs


# 前言
该博客是一个Java搭建的前后端不分离项目，目前还在持续优化中

# 项目介绍
z_blogs是一款简约大气的个人博客，在ui设计上借鉴了网上的很多博客系统，融合了各种元素，在整个系统架构上做了很多优化，
所以当你访问此项目时会感到非常流畅
